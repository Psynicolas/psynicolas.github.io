Featured Project: Course Design Proposal

AI Inventory Assistant Course Proposal

Comprehensive learning design proposal for Make Academy: A mini-course teaching professionals to build AI-powered inventory management systems using no-code tools.

Project Overview

This instructional design project demonstrates a complete course development process from needs analysis to evaluation strategy. The course teaches retail operations managers to create AI agents that automate inventory management through natural language interactions in Slack.

3
Hours Learning
4
Modules

Complete Needs Analysis

Detailed user persona with pain points, learning goals, and preferences based on retail operations managers

Bloom's Taxonomy Alignment

Learning objectives spanning from Understanding to Evaluation, with clear assessment methods

Learner Journey Maps

Program-level and module-level journey maps showing progression from orientation to mastery

Active Learning Focus

Learn-by-doing approach with continuous project building from account setup to customization

Success Metrics

Clear evaluation framework including completion rates, time to first success, and business impact ROI

Cognitive Load Management

Progressive disclosure, chunked content, and just-in-time learning principles throughout

Key Design Elements

  • Real-World Application: Every module builds toward a functional AI inventory assistant
  • Scaffolded Learning: Three-tier progression from observation to guided practice to independent application
  • Multimodal Resources: Interactive demonstrations, GIF tutorials, visual workflows, and hands-on exercises
  • Engagement Strategy: Celebration moments, personal relevance, and immediate business value
  • Assessment Integration: Pre-course diagnostic, module quizzes, and post-course perception survey
